## Root 

```css
:root {
  --spacing-1: 0.5vh;
  --spacing-2: 1vh;
  --spacing-3: 2vh;
  --spacing-4: 5vh;
  --spacing-5: 10vh;
  --spacing-6: 15vh;
  --spacing-7: 20vh;
}
```
